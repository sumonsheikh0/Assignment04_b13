Difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll

getElementById()  selects one element using its unique ID.
getElementsByClassName()  gets all elements with a specific class name.

querySelector() gets the first element that matches a CSS selector.

querySelectorAll()  gets all elements that match a CSS selector.

Event bubbling means when you click on a child element, the event also moves up and triggers events on its parent elements.

answare to the question no 4
============================

Event delegation is when you add one event listener to a parent element instead of adding listeners to each child.It’s useful because it improves performance and also works for elements added later.
